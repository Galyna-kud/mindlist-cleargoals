# MindList — Ваш спосіб продуктивності

**Деплойована версія**  
https://galyna-kud.github.io/mindlist-cleargoals/

## Як запустити локально
Крок 1. Відкрий термінал / командний рядок і перейди в зручну папку, куди хочеш зберегти проєкт(напр., на робочий стіл або в спеціальну папку), командою:
  cd Назва_папки

Крок 2. Клонуй репозиторій: введи цю команду і натисни Enter: 
  git clone https://github.com/Galyna-kud/mindlist-cleargoals.git
  
Крок 3. Зайди всередину папки проєкту командою: 
  cd  mindlist-cleargoals
  
Крок 4. Встанови залежності:
  npm install

Крок 5. Запусти проєкт у режимі розробки:
  npm run dev

**Що має статися:**
Термінал покаже:
  Ready in ...ms
  Local: http://localhost:3000
Якщо порт 3000 зайнятий — просто натисни Y (yes), або запусти з іншим портом:
  npm run dev -- -p 3001
  
Крок 6. Відкрий сайт у браузері:
  1. Відкрий Chrome, Firefox або будь-який браузер.
  2. Встав у рядок адреси: http://localhost:3000
  3. Натисни Enter і зможеш побачити MindList

Крок 7. Якщо хочеш зупинити сервер: у терміналі натисни:
  Ctrl + C → Enter
