# MindList — Ваш спосіб продуктивності

Розумний todo-додаток, який враховує рівень енергії, допомагає входити в потік і не дозволяє вигоріти.  
Pomodoro з адаптацією, XP-гейміфікація, стріки та детальна аналітика.

Деплойована версія:  
https://galyna-kud.github.io/mindlist-cleargoals/
