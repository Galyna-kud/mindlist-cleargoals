## Історія створення MindList

### Основні інструменти
- Grok (xAI) — технічні правки та ідеї назв/описів  
- v0 by Vercel (v0.dev) — генерація початкових компонентів UI, швидке прототипування інтерфейсу та деяких сторінок

### Ключові етапи роботи
1. Початок — попросила створити "кльовий" Todo List на React + TypeScript з localStorage.  
   Додатково вказала головні складові: фільтри, пошук + сказала, щоб згенерував 2 круті фічі(це було встановлення таймера і дашборд аналітики) і 2 було моїх - це стан енергії на виконання завдання і зарахування балів за виконане завдання.  
   v0 згенерував перші гарні компоненти (форми, Pomodoro-таймер).
   
3. Назва та слоган  
   Пробували варіанти: SmartTodo → ZenFlow → PulseMind → FlowCore → зупинилася на MindList з підзаголовком "Ваш спосіб продуктивності".

4. Деплой на GitHub Pages  
   Багато ітерацій: виправляли 404 на _next/static, білий екран, налаштування basePath/assetPrefix, output: 'export', .nojekyll та gh-pages.
